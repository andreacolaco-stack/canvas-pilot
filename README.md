# 🎨 Canvas Pilot
> An agentic bridge between natural language and production-ready SVG geometry.

Canvas Pilot is an open-source **Agentic Vector Engine**. Unlike standard LLMs that hallucinate coordinates, Canvas Pilot uses a geometric constraint engine to ensure every generated SVG is layered, optimized, and developer-friendly.

### 🚀 Key Features
* **Vision-to-Vector:** Upload a napkin sketch; get a clean SVG.
* **Semantic Layering:** AI-named groups (e.g., `<g id="nav-icon">`).
* **Geometric Reasoning:** "Align these circles" or "Make the border 2px thicker."
